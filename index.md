### Current Projects:

1. My first project is Ava (Automated Vigilance Assistant), an AI based around real-world use cases such as identifying open ports, vulnerabilities, log analysis, system scanning and a method to interact with the AI and its tools. This Alpha build is designed to provide the user with a virtual defender hosted locally in their browser. The Ava will be able to converse with the user using free Llama API resources, paired with a simple security information event management (SEIM) interface for the user to track and notate system process, log analysis and real time reporting. Alpha 1.0 and beyond will expand further on core modules and add new modules to enhance/support the Ava's overall security framework.  

2. Adam (Attack Discovery and Modeling), red team ethical attacker. Planning, architecture and knowledge acquisition in progress. Testing Adam will require the use of virtual machines (using Metasploitable) to safety locate vulnerabilities, open ports and plan attacks. This will follow the same version control of Alpha v.1 to v1.0, with modular iterations and docker container. Similarity, Adam will have a similar core module for conversation when providing the user with insights on weak services and how to exploit them.

3. Orion. Custom operating system based on Ubuntu. This is a fun side project to better understand what is happening inside my computer and gain further knowledge. This will be easily tested in the same virtual machines as Ava and Adam. Providing insight to my OS design, and detecting what vulnerabilities exist and how to mitigate them.
